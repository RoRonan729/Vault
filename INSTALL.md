<h1>Installing RoKingdom</h1>

**Tutorial by Ronan Gupta at RoKingdom Servers.**

<h2>Installing on Java (Mac, Linux, and Windows)</h2>

Open Minecraft, then proceed to the Multiplayer button. Next accept the EULA if not done so before, then click Add Server at the bottom of your screen. Next for server name put RoKingdom for server adress type exactly play.rokingdom.com then click done and you will be done and able to join.

<h2>Installing on Bedrock (iOS, ipadOS, Andriod, Minecraft for Windows)</h2>

> [!NOTE]
> Bedrock is more unstable and unreliable the Java! We try our hardest to make you experiece as well as it can be but we do have many issues with Bedrock.
> ALSO Consoles are not supported! This includes Playstation, Xbox, and Switch

To install on bedrock press Play next press the Servers tab then on the left either at the top or the bottom of the list you should see an "Add Server" button. Click it. Once you do fill in name: RoKingdom Server adress: play.rokingdom.com and port: 10594 then click add and play! You can find the server at the bottom of your server list
